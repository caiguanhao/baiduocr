BaiduOCR
========

Read Chinese and English text from JPEG/PNG image with Baidu OCR services.

```go
ocr := baiduocr.OCR{APIKey: APIKey}
results, err := ocr.ParseImageFile("test/fixtures/chinese/hanzi.jpg")
```

See docs for usage and examples.
